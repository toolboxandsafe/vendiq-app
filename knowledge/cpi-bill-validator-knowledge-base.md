# DIAGNOSTIC LED LOCATION

**Series 2000 / CV1000 (AE2xxx, VN2xxx, CV1xxx):**
- Red LED on REAR side, below note magazine
- Flashes error code automatically

**CV3000 / Talos (CV3xxx, T6xx, T7xx, T8xx):**
- Uses bezel arrow lights (front)
- When error occurs, bezel arrows go OFF
- To see error: push a note into slot until it stops, count the flashes

# DIAGNOSTIC ERROR CODES

⚠️ **Clean parts with water only — NO chemical agents!**

| Flash Pattern | Error | Fix |
|---------------|-------|-----|
| **1 flash** | Blocked note path | Remove jams, clean path with damp microfiber cloth |
| **2 flashes** | Problem with other system component | Check controller, inventory, service door closed, coin system, fill coin tubes to 50%+ |
| **3 flashes** | Lower housing assembly problem | Remove lower housing, clean, remove jams/debris, reinstall (listen for latch click) |
| **4 flashes** | Blocked note path | Same as 1 flash — remove jams, clean path |
| **5 flashes** | Note magazine out of position | Remove/reinstall magazine (listen for latch), clean sensor if error continues |
| **Continuous FAST flashes** | Note stacker is FULL | Empty the note magazine |
| **Continuous SLOW flashes** | Note validator FAILURE | Contact CPI — service or replace unit |

# ERROR FIX PROCEDURES

## 1 or 4 Flashes (Blocked Note Path)
1. Remove note magazine — clear any jammed notes
2. Remove lower housing assembly — clear jams, dirt, debris, foreign objects
3. Examine entire note path for blockages
4. Clean all parts with **damp microfiber cloth**
5. Dry parts completely
6. Reassemble

## 2 Flashes (System Component Problem)
1. Check machine controller for errors
2. Check machine inventory — fill products if needed
3. Make sure service door is CLOSED
4. Verify coin payment system works
5. Fill coin tubes to **50% or more**

## 3 Flashes (Lower Housing Problem)
1. Remove lower housing assembly
2. Clear jams, dirt, debris, foreign objects
3. Clean with **damp microfiber cloth**
4. Dry completely
5. Reinstall — should hear **latch click**

## 5 Flashes (Magazine Out of Position)
1. Remove and reinstall note magazine — listen for **latch click**
2. If error continues:
   - Remove magazine
   - Check magazine sensor
   - Clean sensor with **damp microfiber cloth**
   - Dry completely
   - Reinstall — listen for **latch click**

## Continuous Fast Flashes (Stacker Full)
1. Empty the note magazine
2. Done!

## Continuous Slow Flashes (Validator Failure)
1. **Contact CPI** — unit needs service or replacement
2. Phone: 800-628-8363

# 5-FLASH ERROR (Magazine Removed)

**Applies to:** Talos T6/T7/T8, Series 2000, CV1000, CV3000

## How to See the Error

**Series 2000 / CV1000:**
- Red LED on rear (below magazine) flashes 5 times

**Talos / CV3000:**
- Front lights won't be illuminated
- To see error: partially insert a note and hold — bezel arrows flash 5 times

## Possible Causes

1. Note magazine removed or not attached properly
2. Magazine prism missing or damaged
3. Validator and magazine need cleaning
4. Magazine is damaged

## Tools Needed
- Air duster
- Microfiber cloth
- Water only (NO chemical agents!)

## Fix Steps

### Step 1: Check Magazine is Attached
1. Verify latch on top of validator clicked into place
2. Verify magazine notches fully inserted into 4 grooves on validator

### Step 2: Check Magazine Prism
1. Remove magazine (push latch forward, lift and pull back)
2. Check prism is in place and not damaged

### Step 3: Clean Sensor and Prism
1. **Power OFF machine**
2. Remove magazine
3. Wipe validator sensor AND magazine prism with **damp microfiber cloth**
4. Reattach magazine (latch clicks, notches in grooves)
5. Power ON machine

### Step 4: Replace Magazine (if still failing)
1. Remove magazine
2. Try a magazine from another validator that works
3. If error clears = original magazine is bad
4. Contact CPI support for replacement

# VALIDATOR ONLY ACCEPTS FIRST $1 BILL

**Symptom:** Bill validator accepts first $1 note, then stops accepting bills

**Root Cause:** Usually a COIN CHANGER issue, not the bill validator!
- Machine can't make change for additional bills
- Coin tubes are low

## Fix Steps

### Step 1: Check Coin Inventory
1. Open coin changer cassette
2. Make sure nickel, dime, and quarter tubes are **at least half full**
3. (Dollar coin tube doesn't need coins)
4. Power OFF machine, then power ON
5. Test if issue is resolved

### Step 2: Test Coin Acceptance
1. Try 3 separate coin transactions
2. If coins aren't accepted → coin changer problem, contact CPI

### Step 3: Test Coin Dispensing
1. Press coin return on machine
2. Verify coins dispense properly
3. If coins don't dispense → coin changer problem, contact CPI

### Step 4: Check Escrow Setting
1. Verify machine escrow setting isn't causing the issue
2. Contact machine manufacturer if needed

### Step 5: Check Exact Change Condition
1. If machine shows "USE EXACT CHANGE"
2. Try replacing coin mech with known working unit
3. Fill nickel/dime/quarter tubes at least half full
4. Test again
